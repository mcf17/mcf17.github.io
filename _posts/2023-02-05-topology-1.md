## 1.1 위상공간이란?

주어진 집합에 위상을 정의하는 방법은 여러가지가 있으나, **열린 부분집합**을 정의하는 방법이 보편적이다. 

{% capture notice-1 %}

### Definition 1.

주어진 집합 $X$ 상의 **위상** (**topolopgy** on $X$)은 다음 세 조건을 만족하는 $X$의 부분집합족 $\mathcal O \subseteq \mathcal P(X)$으로 정의한다.

1. $\emptyset, X \in \mathcal O$.
2. $\mathscr U \subseteq \mathcal O \implies \bigcup \mathscr U \in \mathcal O$,
3. $U,V \in \mathcal O \implies U\cap V \in \mathcal O$.

그리고 $X$와 $\mathcal O$를 순서쌍으로 묶은 $(X,\mathcal O)$를 **위상공간** (**topological space**)이라 정의한다. 

{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>


{% capture notice-1 %}

### Definition 2.
위상공간 $(X,\mathcal O)$에 대해 $\mathcal O$에 속한 집합들을 $(X,\mathcal O)$의 열린부분집합 (**open subsets of $(X,\mathcal O)$**)으로 정의한다. 

{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>

**Definition 1**에서 확인해야 할 점은 크게 세 가지다. 

- 조건 (1)에 의해 $\mathcal O$는 무조건 $\emptyset$과 $X$를 포함하므로 공집합이 될 수 없다. 즉 $\emptyset$과 $X$은 항상 $(X,\mathcal O)$의 열린부분집합이다.
- 조건 (2)에 의해 열린부분집합들의 합집합은 다시 열린부분집합이 된다. 
- 조건 (3)에 수학적 귀납법을 적용하면, 열린부분집합들의 **유한**교집합은 다시 열린부분집합이 된다. 조건 (2)와 다르게 교집합의 경우 **유한번의 연산에 대해서만 닫혀있다**. 

**Definition 2**에서의 "열려있다"의 개념은 **주어진 위상공간에 따라 상대적**이다. 문맥상 명확한 경우가 아니라면 뒤의 위상공간을 생략하면 안된다. 


| $U$가 $X$의 부분집합 | $\mathcal O$  | 용례                                                         |
| ---------------------- | ------------- | ------------------------------------------------------------ |
| 명확하지 않음          | 명확하지 않음 | $U$는 $(X,\mathcal O)$의 열린부분집합이다.                   |
| 명확하지 않음          | 명확함        | $U$는 $X$의 열린부분집합이다. ($U$ is an open subset of $X$) |
| 명확함                 | 명확하지 않음 | $U$는 $(X,\mathcal O)$에서 열려있다. ($U$ is open in $(X,\mathcal O)$) |
| <span style="color: #088A68"> 명확함 </span>                 | <span style="color: #088A68"> 명확함 </span>         | <span style="color: #088A68"> $U$는 $X$에서 열려있다. ($U$ is open in $X$)     </span>           |

대부분 둘 다 명확하므로 마지막 표현이 사실상 표준이 된다.

## 1.2 위상의 예시

집합론만을 사용해 정의 가능한 위상의 예시는 다음과 같다.

{% capture notice-1 %}

### Example 3.

1. $\mathcal O = \\{ \emptyset, X \\}$. 가장 간단한 예시로 **자명한 위상** (trivial topology) 또는 **비이산위상** (indiscrete topology)이라고 한다.
2. $\mathcal O = \mathcal P(X)$. 멱집합 자체도 $X$의 위상이 될 수 있다. 이를 **이산 위상** (discrete topology)이라고 한다.

{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>

**드모르간의 법칙** (DeMorgan's law)를 사용하면 다음 두 예시도 위상임을 보일 수 있다.

{% capture notice-1 %}

### Example 4.

1. $\mathcal O = \\{ \emptyset \\} \cup \\{ U \subseteq X : X\setminus U \text{ is finite} \\}$. 여집합이 유한집합이 되는 집합들을 모아놓았기 때문에 이름이 **여위상집합** (cofinite topology). 
2.  $\mathcal O = \\{ \emptyset \\} \cup \\{ U \subseteq X : X\setminus U \text{ is countable} \\}$. 여집합이 가산집합이 되는 집합들을 모아놓았기 때문에 이름이 **여가산집합** (cocountable topology). 

1은 $X$가 무한집합일 경우를, 2는 $X$가 비가산집합일 경우에 대비해 $\emptyset$를 명시적으로 $\mathcal O$에 포함시켜 놓는다.

{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>

{% capture notice-1 %}

***Proof of Example 4***. 

1. $\emptyset$은 정의에 의해 $\mathcal O$에 속해있고, $X\setminus X = \emptyset$으로 유한하므로 $X \in \mathcal O$를 만족한다. 
2. 유한 (resp. 가산) 집합의 모임 $\\{U_\alpha \\}$에 대해, $ X \setminus \left( \bigcup_\alpha U_\alpha \right) = \bigcap_\alpha (X\setminus U_\alpha)$ 로 표현할 수 있다. 결국 유한 (resp. 가산) 집합들의 교집합이므로 유한 (resp. 가산)이다. 
3. 두 유한 (resp. 가산) 집합 $U, V$에 대해, $X \setminus (U\cap V) = (X\setminus U) \cup (X\setminus V)$ 이므로 유한 (resp. 가산)이다.

결론적으로 **Definition 1**의 조건을 모두 만족하여 둘 다 위상이 맞다.

{% endcapture %}

<div class="notice">{{ notice-1 | markdownify }}</div>


## 1.3 위상의 비교

앞서 보았듯이 주어진 집합 $X$에 여러가지 위상을 정의할 수 있다. 위상은 멱집합의 부분집합이라는 점을 떠올려본다면, 동일한 집합에 주어진 서로 다른 위상을 **집합의 포함관계** (subset relation) $\subseteq$를 이용하여 비교가능하다. 

{% capture notice-1 %}
### Definition 5. 
집합 $X$에 주어진 두 위상 $\mathcal O_1, \mathcal O_2$에 대해 $\mathcal O_1 \subseteq \mathcal O_2$를 만족할 경우 
1. $\mathcal O_1$은 $\mathcal O_2$보다 **거친 위상** (coarser topology)이라 하고 
2. 반대로 $\mathcal O_2$는 $\mathcal O_1$보다 **섬세한 위상** (finer topology)라 한다. 

{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>

물론 **Definition 5**를 적용할 수 없는 경우도 존재한다. 양쪽 모두 다른 한쪽을 포함하지 않는 다면, 이를 **비교불가능**한 경우로 본다 (incomparable). 

**Definition 5**를 **Example 4**에 적용하면 다음을 얻는다. 

{% capture notice-1 %}

### Observation 6.
$X$위 비이산위상을 $\mathcal O_{ind}$, 이산위상을 $\mathcal O_{disc}$, 여유한위상을 $\mathcal O_{cofin}$, 여가산위상을 $\mathcal O_{cocnt}$라 하면, 다음이 성립한다. 

1. $\mathcal O_{ind} \subseteq \mathcal O_{cofin} \subseteq \mathcal O_{cocnt} \subseteq \mathcal O_{disc}$.
2. $X$가 원소의 개수가 2개 이상인 유한집합이라면 ($1 < \| X \| < \infty$) $\mathcal O_{ind} \subsetneq \mathcal O_{cofin} = \mathcal O_{cocnt} = \mathcal O_{disc}$.
3. $X$가 가산무한집합이라면 (countably infinite) $\mathcal O_{ind} \subsetneq \mathcal O_{cofin} \subsetneq \mathcal O_{cocnt} = \mathcal O_{disc}$.
4. $X$가 비가산집합이라면 $\mathcal O_{ind} \subsetneq \mathcal O_{cofin} \subsetneq \mathcal O_{cocnt} \subsetneq \mathcal O_{disc}$.

{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>


{% capture notice-1 %}
***Proof of Observation 6***.  
1. 유한집합은 가산집합이므로 $\mathcal O_{cofin} \subseteq \mathcal O_{cocnt}$.
2. 전체집합 $X$가 유한집합이라면, 그 어떤 부분집합 $U$를 가져와도 여집합 $X\setminus U$는 유한집합이 된다. 따라서 $\mathcal O_{cofin} = \mathcal O_{cocnt} = \mathcal P(X) = \mathcal O_{disc}$를 얻는다. 원소의 개수가 2개 이상이므로, 임의의 원소 $x\in X$에 대해 $\\{ x \\} \neq X$이고 $\\{x\\}$는 $(X,\mathcal O_{cofin})$에서 열려있다. 따라서 $\mathcal O_{ind} \subsetneq \mathcal O_{cofin}$을 얻는다
3. (2)와 비슷한 논리를 적용하면 된다. 
4. (3)과 비슷한 논리를 적용하고, 비가산집합에서 가산집합을 빼도 비가산집합임을 이용하여 $\mathcal O_{cocnt} \subsetneq \mathcal O_{disc}$를 보여주면 완료. 
{% endcapture %}

<div class="notice">{{ notice-1 | markdownify }}</div>

다시 돌아가 집합 $X$ 상에 정의된 위상들의 모임 $\\{ \mathcal O_\alpha \\}$이 주어졌다고 가정하면 크게 두 가지 질문을 던질 수 있다.

1. 모든 $\mathcal O_\alpha$보다 거친 위상 중 **가장 섬세한 위상**의 존재 (finest topology contained in every $\mathcal O_\alpha$)
2. 모든 $\mathcal O_\alpha$보다 섬세한 위상 중 **가장 거친 위상**의 존재 (coarsest topology that contains every $\mathcal O_\alpha$)
   
1번의 경우 모든 $\mathcal O_\alpha$의 교집합

$$
\mathcal O = \bigcap_\alpha \mathcal O_\alpha
$$

이 답이다. 증명하자면, 
- 모든 $O_\alpha$가 $\emptyset, X$를 포함하고 있으므로 $\emptyset, X \in \mathcal O$를 만족한다.
- 임의의 부분집합족 $\mathscr U \subseteq \mathcal O$는 모든 $\alpha$에 대해 $\mathscr U \subseteq \mathcal O_\alpha$를 만족한다. 모든 $\mathcal O_\alpha$는 합집합 연산에 닫혀있으므로 $\bigcup \mathscr U \in \mathcal O$를 만족한다.
- 동일한 논리로 $\mathcal O$는 **유한**교집합 연산에 닫혀있다. 

따라서 $\mathcal O$는 $X$위의 위상이다. 모든 $\mathcal O_\alpha$보다 거친 임의의 위상 $\mathcal T$는 항상 $\mathcal T \subseteq \bigcap_\alpha \mathcal O_\alpha = \mathcal O$를 만족하므로, $\mathcal O$는 그러한 위상들 중 가장 섬세하다. 

{% capture notice-1 %}
### Proposition 7. 

집합 $X$ 상에 정의된 위상들 $\\{ \mathcal O_\alpha \\}$보다 거친 위상 중 가장 섬세한 위상은 

$$
\mathcal O = \bigcap_\alpha \mathcal O_\alpha 
$$

이다. 

{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>

아쉽게도 2번의 경우 $\mathcal O = \bigcup_\alpha \mathcal O_\alpha$는 답이 아니다. 이유는 합집합 공리를 항상 만족하지는 않기 때문이다. $\mathcal O_1$의 원소 $U$와 $\mathcal O_2$의 원소 $V$가 있을 때, $U\cup V$가 항상 $\mathcal O_1 \cup \mathcal O_2$의 원소라고 말할 수 있을까? 조금만 생각해봐도 그건 아니다. 

{% capture notice-1 %}
### Counter Example 8. 

$X = \\{1,2,3\\}$, $\mathcal O_1 = \\{ \emptyset, X, \\{ 1 \\} \\}$, $\mathcal O_2 = \\{ \emptyset, X, \\{2\\}  \\}$

{% endcapture %}

<div class="notice">{{ notice-1 | markdownify }}</div>

2번은 1번의 결론을 적절히 이용하여 답하면 된다. 임의의 위상이 모든 $O_\alpha$보다 섬세할 필요충분조건은 $\bigcup_\alpha O_\alpha$를 포함하는 것이다. 그 중 가장 거친 위상은 1번을 이용하면 다음과 같다. 

$$
\mathcal O' = \bigcap \left\{ \mathcal O : \mathcal O \text{ is a topology on } X \text{ and } \bigcup_\alpha \mathcal O_\alpha \subseteq \mathcal O   \right\}
$$

마지막으로 우변의 집합이 공집합이 아님을 보여야 한다. 이는 이산 위상 $\mathcal O_{disc} =\mathcal P(X)$의 존재로 해결된다! 2번에 대한 대답은 나중에 나올 부분기저 (sub-base)와 기저 (base)의 중요한 모티브가 된다. 

1번과 2번 물음에 대한 대답을 종합하면,

{% capture notice-1 %}
### Proposition 9. 

주어진 집합 $X$에 정의할 수 있는 모든 위상의 모임을 $\mathcal O(X)$라 하고 $\mathcal O(X)$위에 이항관계 $\leq$를 다음과 같이 정의하면, 

$$
\mathcal O_1 \leq \mathcal O_2 \iff \mathcal O_1 \subseteq \mathcal O_2
$$

$\big( \mathcal O(X), \leq \big)$는 [**완비격자** (complete lattice)](https://en.wikipedia.org/wiki/Complete_lattice)이다. 
{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>

{% capture notice-1 %}
***Proof of Proposition 9***. 부분족 $\mathscr O = \\{\mathcal O_\alpha\\} \subseteq \mathcal O(X)$가 주어졌을 때, 
1. $\bigcap_\alpha \mathcal O_\alpha = \inf \mathscr O$,
2. $\bigcap \\{ \mathcal O : \mathcal O \text{ is a topology on } X \text{ and } \bigcup_\alpha \mathcal O_\alpha \subseteq \mathcal O   \\} = \sup \mathscr O$
로 주어진다. 

$\mathscr O$가 공집합일 경우, supremum (join)은 비이산위상 $\mathcal O_{ind}$, infimum (meet)은 이산위상 $\mathcal O_{disc}$로 주어진다. 
{% endcapture %}
